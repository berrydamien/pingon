# pingon

This script is use for ping a machine (server) diring the reboot and he shows you when the machine is online.

## Installation

Copy in your bin path (~/bin , /usr/local/bin/) and use !

## Exemple

`$ pingon 192.168.1.2

Waiting ...                  /

`

`$ pingon 192.168.1.2

Waiting ...                  [OK]

`
